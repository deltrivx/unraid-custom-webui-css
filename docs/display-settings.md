# Display Settings

After installing the theme, open Unraid **Settings -> Display Settings**.

Set the main WebGUI font colors used by the dark layout to white or near-white. This keeps table text, module titles, and navigation labels readable over the custom background.

Recommended values:

```text
Page text color: #ffffff
Header text color: #ffffff
Menu text color: #ffffff
```

The exact field names may vary slightly between Unraid versions and installed plugins. The goal is to keep the three primary text color settings white.
